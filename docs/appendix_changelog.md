# Appendix B: 変更履歴

## v2.0.0 (2026/02/23) — GitHub公開版

### 主要な変更
- QIIME 2 バージョンを **2019.7 → 2026.1** に更新
- リファレンスデータベースを **SILVA 132 → SILVA 138.2** に更新
- **RESCRIPt プラグイン**による分類器作成手順を追加（推奨方法として記載）
- インストール方法を更新（VirtualBox → **WSL2**、Apple Silicon対応追加、**Docker**対応追加）
- **R / phyloseq へのエクスポート**セクションを新規追加（qiime2R パッケージ）
- **組成データ解析**の注意事項を追加（ANCOM-BC, ALDEx2, CLR変換）
- **トラブルシューティング**セクションを新規追加
- マニュアルをMarkdownに変換し、GitHub上でセクション別に閲覧可能に
- Mermaid記法によるフローチャートを追加
- scikit-learn バージョン不一致に関する注意書きを更新（現行バージョンでは不要）
- GreenGenes は非推奨として明記（GreenGenes2 の存在にも言及）
- DADA2パラメータが固定値ではなくランごとに確認すべきであることを強調
- shell scriptのforループを活用したバッチ処理を一部導入

### 保持した内容
- 内部標準（IS）を用いた絶対定量ワークフロー（検量線・LINEST関数）
- マルチラン DADA2 → merge の正しい手順
- ネガティブコントロール除去ワークフロー
- メタデータファイルの詳細仕様
- Confidence フィルタリングの詳細解説
- 各コマンドのパラメータ説明・挙動の解説
- Provenance tracking の解説
- ファイル名一括変更（rename.R）
- タグ付きサンプルの処理手順
