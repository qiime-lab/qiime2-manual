## v2.0.0 (2026/02/23) — GitHub 公開版

### 主要な変更
- QIIME 2 バージョンを **2019.7 → 2026.1** に更新
- リファレンスデータベースを **SILVA 132 → SILVA 138.2** に更新
- **RESCRIPt プラグイン**による分類器作成手順を追加（推奨方法として記載）
- インストール方法を更新（VirtualBox → **WSL2**、Apple Silicon 対応追加、**Docker** 対応追加）
- **R / phyloseq へのエクスポート**セクションを新規追加（qiime2R パッケージ）
- **組成データ解析**の注意事項を追加（ANCOM-BC, ALDEx2, CLR 変換）
- **トラブルシューティング**セクションを新規追加
- マニュアルを Markdown に変換し、GitHub 上でセクション別に閲覧可能に
- Mermaid 記法によるフローチャートを追加
- scikit-learn バージョン不一致に関する注意書きを更新（現行バージョンでは不要）
- GreenGenes は非推奨として明記（GreenGenes2 の存在にも言及）
- DADA2 パラメータが固定値ではなくランごとに確認すべきであることを強調
- shell script の for ループを活用したバッチ処理を一部導入
